## ü§ñ AI OVERVIEW OPTIMIZATION

### **Wat is AI Overview?**
Google's AI-generated answer box bovenaan search results.

### **Hoe rank je in AI Overview?**

**1. Featured Snippet Format**
```html
<h2>Wat is [Topic]?</h2>
<p><strong>Direct antwoord in 40-60 woorden.</strong></p>
<p>Uitgebreide uitleg...</p>
```

**2. FAQ Schema**
```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "Wat is Google Core Update?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Een Google Core Update is..."
    }
  }]
}
```

**3. List Format**
```html
<h3>5 Stappen om [Goal] te bereiken:</h3>
<ol>
  <li><strong>Stap 1:</strong> Concrete actie</li>
  <li><strong>Stap 2:</strong> Concrete actie</li>
  ...
</ol>
```

**4. Table Format**
```html
<table>
  <tr>
    <th>Feature</th>
    <th>Description</th>
    <th>Impact</th>
  </tr>
  ...
</table>
```

**5. Expert Quotes**
```html
<blockquote>
  "Expert statement over topic" - John Mueller, Google
</blockquote>
```
