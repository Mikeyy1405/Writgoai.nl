## üîå MCP (Model Context Protocol) Integration

### Wat is MCP?

MCP is een **open standaard** ge√Øntroduceerd door Anthropic (november 2024) en geadopteerd door OpenAI en Google DeepMind.

### Wat kan het?

MCP laat AI agents **direct verbinden** met:
- **Tools:** GitHub, Google Tasks, Google Maps, Figma
- **APIs:** REST APIs, GraphQL
- **Databases:** SQL, MongoDB, Redis
- **Services:** Slack, Jira, externe services

### Voordelen

- ‚úÖ **Real-time context** - Live data van je tools
- ‚úÖ **Actionable automation** - Agent kan daadwerkelijk acties uitvoeren
- ‚úÖ **Geen custom integraties** - Standaard protocol
- ‚úÖ **Eenvoudige configuratie** - JSON-based server setup

### Abacus.AI MCP Support

**Producten met MCP:**
1. **Abacus AI Desktop** - AI desktop assistant met tool extensions
2. **Deep Agent** - Proto AGI met tool access
3. **CodeLLM** - AI coding assistant met API/database toegang

**Limieten:**
- Tot **5 servers** tegelijk
- Tot **50 tools** across servers
- Zowel **local** (npm/pypi) als **remote servers**
