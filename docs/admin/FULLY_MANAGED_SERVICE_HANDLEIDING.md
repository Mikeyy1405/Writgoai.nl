## Veelgestelde Vragen

### Voor Klanten

**Q: Kan ik meerdere projecten fully managed laten beheren?**
A: Ja! Elk project kan afzonderlijk worden overgedragen. Je betaalt â‚¬199 per project per maand.

**Q: Behoud ik toegang tot mijn content?**
A: Absoluut! Je hebt volledige read-only toegang en kunt alle content bekijken en downloaden.

**Q: Kan ik nog steeds zelf content toevoegen?**
A: Bij fully managed projecten verzorgt Writgo alle content. Als je zelf content wilt toevoegen, kun je de service opzeggen en terug naar zelfbeheer gaan.

**Q: Hoe lang duurt het voordat content wordt gepubliceerd?**
A: We starten binnen 24 uur na activatie met de eerste content planning en publicatie.

**Q: Kan ik de service tussentijds opzeggen?**
A: Ja, de service is maandelijks opzegbaar zonder lange termijn verplichtingen.

**Q: Wat gebeurt er met mijn bestaande content?**
A: Al je bestaande content blijft behouden. We bouwen verder op wat er al is.

### Voor Writgo Team

**Q: Hoe weet ik welke projecten nieuwe content nodig hebben?**
A: Check wekelijks het admin dashboard. Sorteer op laatste update datum.

**Q: Kan ik content genereren zonder credits te gebruiken van de klant?**
A: Ja, bij fully managed projects komen de credits van het Writgo admin account (unlimited).

**Q: Wat als een klant specifieke wensen heeft?**
A: Pas de project instellingen aan (target audience, brand voice, keywords) en gebruik custom instructions.

**Q: Hoe vaak moet ik content publiceren?**
A: Richtlijn: minimaal 4-8 blogs per maand per project, plus social media content.


- **POST** `/api/client/projects/transfer-management`

  - Gebruikt door klanten om service te beheren

- **GET** `/api/client/projects/transfer-management`
  - Voor klanten: eigen managed projects
