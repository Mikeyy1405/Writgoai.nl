Alle blog posts zijn gekoppeld aan een specifiek project:

Social media strategie en posts zijn per project:

### Website Analyse

Website analyse gebruikt het huidige project:

- **Niche detectie** per website

### Topical Authority Maps

- **100-500 artikelen** per project
- **SEO structuur** per website
- **Keyword clustering** project-specifiek

## FAQ

### Hoeveel projecten kan ik aanmaken?

Er is geen limiet aan het aantal projecten. Je kunt zoveel websites beheren als je wilt.

### Kan ik content verplaatsen tussen projecten?

Momenteel niet. Content is permanent gekoppeld aan het project waarin het is aangemaakt.

### Wat gebeurt er als ik een project verwijder?

âš ï¸ **Let op:** Alle content (blog posts, social media posts, analyses) wordt ook verwijderd. Dit kan niet ongedaan worden gemaakt.

### Kan ik een project tijdelijk deactiveren?

Ja! Zet de status op "Inactief" in plaats van het project te verwijderen. De content blijft behouden.

### Hoe weet ik welk project actief is?

- De **Project Switcher** toont het huidige project
- Het actieve project heeft een âœ“ vinkje
- Alle pagina's tonen content van het actieve project

### Moet ik handmatig switchen bij elke pagina?

Nee! Het huidige project blijft actief over alle pagina's heen tot je handmatig switcht.

### Kan ik meerdere projecten tegelijk bewerken?

Nee, je kunt slechts Ã©Ã©n project tegelijk actief hebben. Switch tussen projecten om verschillende websites te beheren.

### Wat als ik een project per ongeluk verwijder?

Helaas kunnen verwijderde projecten niet worden hersteld. Wees voorzichtig bij het verwijderen en zorg voor backups van belangrijke content.
